UniCalc Scholar is a professional, multi-year CGPA (Cumulative Grade Point Average) calculator designed specifically for the Nigerian university grading system. It allows students to manage their academic records across multiple years and export a formal transcript in PDF format.

Key Features:
* Multi-Year Support: Add as many academic years as needed (100L, 200L, etc.) to calculate a lifetime CGPA.

* Real-time Dashboard: A 3-column summary display showing Total Credit Units, Semester-specific GPAs, and Cumulative CGPA.

* Responsive Design: Fully optimized for mobile screens. Tables are scrollable horizontally to ensure data entry is easy on any device.

* PDF Export: Generate a clean, structured academic report including student bio-data and a breakdown of all courses and grades.

* Smart Classification: Automatically determines your degree class (e.g., First Class, Second Class Upper) based on standard 5.0 CGPA scales.

Built With:
* HTML5 & CSS3: For a clean, modern, card-based user interface.

* JavaScript (Vanilla): For all mathematical calculations and dynamic year/row generation.

* jsPDF & AutoTable: To handle the generation of professional PDF transcripts.

* Inter Fonts: For high readability and a professional look.

How to Use:
* Enter Bio-Data: Start by filling in your Name and Matriculation Number.

* Input Courses: For each semester, enter the Course Code, Credit Units, and select your Grade.

* Add Years: If you are in a higher level, click the "+ Add Another Academic Year" button to expand your record.

* Export: Click "Download Full Transcript PDF" to save a copy for your records or applications.
